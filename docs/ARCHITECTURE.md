### State Descriptions

| State | Code | Entry Condition | Exit Condition |
|-------|------|-----------------|----------------|
| BOOT | 0x00 | Power on | Hardware initialized |
| INIT | 0x01 | From BOOT | EEPROM loaded |
| IDLE | 0x02 | Ready for commands | Command received |
| CALIBRATING | 0x03 | 'cal' command | Calibration complete |
| MOVING_OPEN | 0x04 | 'open' command | Limit reached |
| MOVING_CLOSE | 0x05 | 'close' command | Limit reached |
| POSITIONING | 0x06 | Position command | Time elapsed |
| HOLDING | 0x07 | At target position | Timeout |
| SLEEP | 0x08 | 30s idle | Activity or timeout |
| HIBERNATE | 0x09 | 5min in sleep | Button press |
| FAULT | 0x0A | Any error | Recovery or reset |
| RECOVERING | 0x0B | Auto-recovery | Recovery complete |

## Interrupt Handling

### Interrupt Priority

| Priority | Source | Handler | Latency |
|----------|--------|---------|---------|
| Highest | Reset | N/A | N/A |
| High | IWDG | Auto-reset | <1.7s |
| Medium | UART RX | RX ISR | <10µs |
| Low | Timer | millis() | <100µs |

### Critical Sections

```c
/* Disable interrupts for atomic operations */
__asm__("sim");  /* Set interrupt mask */

/* Critical code here */
volatile_operation();

__asm__("rim");  /* Reset interrupt mask */
```
